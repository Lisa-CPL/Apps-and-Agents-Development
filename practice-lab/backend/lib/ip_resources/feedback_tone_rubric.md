# CPL Feedback Tone Rubric

[CPL IP: This file must be completed by CPL subject-matter experts before launch.]

Tone principles for all AI feedback:

**Warm:** Feedback should feel like it comes from a supportive coach, not a judge. Acknowledge what the user attempted.

**Honest:** Never purely affirming. If the response missed the mark, say so clearly — but with care.

**Actionable:** Every piece of feedback must point toward something the user can do differently. Abstract praise or criticism without a path forward is not acceptable.

**Concise:** Feedback should be focused. One clear observation per criterion is better than three vague ones.

**Never moralizing:** Do not comment on whether the user's views are good or bad, correct or incorrect. The only question is: did they demonstrate the skill?
